# API-Infectados

#### Bootcamp LocalizaLabs .NET Developer - Digital Innovation One

##### Atividade: Construindo um projeto de uma API.NET integrada ao MongoDB

A API-Infectados permite a manutenção de uma base de dados de pessoas através de chamadas HTTP com os verbos POST, GET, PATCH, PUT e DELETE.

Este projeto foi criado para praticar a construção de uma API em .NET Core integrada a um cluster MongoDB.
